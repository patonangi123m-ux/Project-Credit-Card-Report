# Project-Credit-Card-Report

# Project Explanation
Membuat sebuah dashboard dari data dummy sebuah perusahaan di Amerika Serikat yang menampung data kartu kredit nasabah.

Beberapa Key Performance Index utama yang ada di dalam dashboard adalah:
1. Number of Customer: jumlah customer atau nasabah.
2. Total Transaction Volume: volume transaksi yang dilakukan oleh nasabah.
3. Total Transaction Amount: jumlah transaksi yang dilakukan oleh nasabah.
4. Total Delinquent Account: jumlah akun nasabah yang menunggak tagihan.
5. Interest Earned: bunga yang diperoleh Bank.
6. Average Customer Satisfaction: tingkat kepuasan nasabah.
7. Average Utilization Ratio: rasio rata-rata penggunaan kartu kredit.

Selanjutnya, kita akan membagi dashboard menjadi:
1. Customer Demographic: persebaran demografi nasabah.
2. Customer Revenue Profile: melihat profil nasabah yang paling menguntungkan.
3. Customer Risk Profile: profil resiko nasabah.
